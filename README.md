# AnimationUtils
>使用VectorDrawable、贝塞尔曲线(二阶、三阶)、PathMeasure以及属性动画打造市面上热门动画效果。
>eg:注水(水波纹)动画、饿了么加载动画、直播点赞动画、QQ空间圣诞点赞动画等。

/**************************持续更新中***************************/<br><br>
